    Details

    Get inspired by the 40-year Heritage of JanSport with the Hipster Canvas Edition duffel bag. A reissue of an original style from 1982, the bag features the original 1967 JanSport woven logo, antique brass details and seat belt webbing straps.

    Features

    • Dual side pockets
    • Large U-shape opening
    • One large main compartment with antique brass metal zippers
    • Original JanSport woven label from 1967 Original weather canvas fabric
    • Removable seat belt webbing shoulder strap
    • Seat belt webbing handles
